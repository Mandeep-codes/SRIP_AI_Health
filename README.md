## Quick Start (Mac / Linux)

```bash
# 1. Clone / enter the project
cd SRIP_AI_Health

# 2. Create a virtual environment (Python 3.10+ recommended)
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Generate synthetic data (creates Data/AP01 вЂ¦ AP05)
python scripts/generate_data.py -out_dir Data

# 5. Visualise one participant в†’ saves PDF to Visualizations/
python scripts/vis.py -name Data/AP01

# 6. Build the labelled feature dataset в†’ saves to Dataset/
python scripts/create_dataset.py -in_dir Data -out_dir Dataset

# 7. Train & evaluate with LOPO cross-validation в†’ saves to Results/
python scripts/train_model.py -dataset Dataset/breathing_dataset.csv

# Optional: use ConvLSTM instead of CNN
python scripts/train_model.py -dataset Dataset/breathing_dataset.csv --model conv_lstm
```

## Methods

### Signal Preprocessing (`create_dataset.py`)
- **Bandpass filter** (0.17 вЂ“ 0.40 Hz) applied to nasal airflow and thoracic movement using a 4th-order Butterworth IIR filter (forward-backward via `sosfiltfilt` for zero phase distortion).
- **Low-pass filter** (< 0.5 Hz) applied to SpOв‚‚ for smoothing.
- **30-second windows** with **50% overlap** are extracted.
- **Labelling rule**: a window receives an event label if > 50% of the window duration overlaps with an annotated event (Obstructive Apnea / Hypopnea); otherwise "Normal".

### Feature Extraction
For each window, 26 features are extracted:
- Nasal & thoracic: mean, std, abs-max, P25/P75, skewness, kurtosis, mean absolute difference, breathing-band power ratio, breathing-band absolute power.
- SpOв‚‚: mean, std, min, range (desaturation depth proxy), trend.
- Cross-signal: NasalвЂ“Thoracic Pearson correlation.

### Models
| Model | Architecture |
|---|---|
| `cnn` | 4Г— Conv1DвЂ“BNвЂ“ReLUвЂ“Dropout в†’ AdaptiveAvgPool в†’ 2Г— FC |
| `conv_lstm` | 3Г— Conv1DвЂ“BNвЂ“ReLUвЂ“MaxPool в†’ BiLSTM (2 layers) в†’ Self-Attention в†’ FC |

### Evaluation
- **Leave-One-Participant-Out (LOPO)** cross-validation (5 folds for 5 participants).
- Metrics reported per fold and averaged: **Accuracy, Precision, Recall, F1** (macro).
- **Confusion matrix** per fold and overall.
- Class-weighted loss to handle label imbalance.
- Early stopping with patience = 8 epochs.
